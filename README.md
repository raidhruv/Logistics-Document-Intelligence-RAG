# Logistics-Document-Intelligence-RAG

## Overview

An AI-powered document intelligence system that uses **Retrieval-Augmented Generation (RAG)** to analyze logistics documents and answer queries with context-aware, evidence-backed responses. It transforms unstructured documents into a searchable knowledge base.

---

## Features

*  Document ingestion (PDF, text, etc.)
*  Semantic search using embeddings
*  Context-aware Q&A with LLMs
*  Source-based responses (RAG)
*  Fast retrieval with vector database

---

## Tech Stack

Python • Streamlit • LLMs • Vector DB (FAISS/Chroma)

---

## Usage

```bash
git clone https://github.com/raidhruv/Logistics-Document-Intelligence-RAG
cd Logistics-Document-Intelligence-RAG
pip install -r requirements.txt
streamlit run app.py
```

---

## How it Works

Upload Documents → Embed & Store → Retrieve Relevant Chunks → Generate Answer

## Use Cases

* Logistics document analysis
* Supply chain insights
